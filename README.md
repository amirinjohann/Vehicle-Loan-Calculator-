# Vehicle-Loan-Calculator-

Simple Vehicle Loan Calculator using Android Studio

A simple and stylish Android application designed to help users quickly estimate their monthly vehicle loan payments.This application provides a clean, user-friendly interface where users can input the total vehicle price, down payment amount, loan period in years, and the annual interest rate. It then calculates and displays a detailed breakdown of the loan, including the principal loan amount, total interest payable, the total repayment amount, and the estimated monthly installment.

Features:
•Calculate vehicle loan details based on user input.
•A dedicated, easy-to-read results page.
•Modern, centered UI with custom-styled buttons.
•An "About" page with author and project information.
•Built using modern Android development practices with layouts that support various screen sizes.
